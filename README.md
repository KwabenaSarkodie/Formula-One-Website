# Formula-One-Website
A front-end web project that recreates a Formula 1 information platform which includes the names and images of drivers,their team details,their nicknames and a peacial "Did you know" feature of each driver.
 The goal of the project is to practice front-end web development concepts such as responsive layouts, semantic HTML structure, and CSS styling.

## Technologies Used
- HTML5
- CSS3
- JavaScript (if used)

## Features
- Driver and team profiles
- "Did you know" section
- Responsive layout
- Modern F1-inspired design
